## 一、产品概述

### 1.1 产品背景

加拿大移民申请流程复杂，评分体系多样，申请人需要手动查阅多个官方网站才能了解自己的分数状态。目前市场上缺乏一款全面、准确、中文友好的综合计算工具。本产品旨在为华人移民申请者提供一站式的分数计算与历史数据查询平台。

### 1.2 产品目标

- **主要目标**：提供准确的联邦 EE、BCPNP、OINP 三大移民项目分数计算
- **次要目标**：展示近一年各项目邀请历史数据，帮助用户判断申请时机
- **用户体验目标**：纯中文界面，操作简单直观，无需注册登录

### 1.3 目标用户

| 用户群体 | 描述 | 核心需求 |
|---------|------|---------|
| 移民申请者 | 正在准备或已提交申请的华人 | 了解自己的分数、判断是否达到邀请门槛 |
| 移民顾问 | 帮助客户规划申请路径 | 快速计算并比较不同场景下的分数 |
| 研究者/学生 | 了解加拿大移民政策 | 学习了解各项目评分规则 |

### 2.2 F1 — 联邦 EE (CRS) 分数计算器

#### 2.2.1 功能描述

实现完整的 CRS（综合排名系统）评分计算，满分 1200 分。

#### 2.2.2 评分结构

```
CRS 总分 (最高 1200 分)
├── A. 核心人力资本因素 (最高 500/460 分)
│   ├── 年龄 (最高 110/100 分)
│   ├── 学历 (最高 150/140 分)
│   ├── 第一官方语言 (最高 136/128 分)
│   ├── 第二官方语言 (最高 24 分)
│   └── 加拿大工作经验 (最高 80/70 分)
├── B. 配偶/同居伴侣因素 (最高 40 分)
│   ├── 配偶学历 (最高 10 分)
│   ├── 配偶第一官方语言 (最高 20 分)
│   └── 配偶加拿大工作经验 (最高 10 分)
├── C. 技能可转移因素 (最高 100 分)
│   ├── 学历 + 语言 (最高 50 分)
│   ├── 学历 + 加拿大工作经验 (最高 50 分)
│   ├── 海外工作经验 + 语言 (最高 50 分)
│   ├── 海外工作经验 + 加拿大工作经验 (最高 50 分)
│   └── 技术资质证书 + 语言 (最高 25 分)
└── D. 附加分项 (最高 600 分)
    ├── 省提名 (600 分)
    ├── 工作 Offer (200/50 分)
    ├── 加拿大学历 (15/30 分)
    ├── 法语能力 (25/50 分)
    └── 在加兄弟姐妹 (15 分)
```

#### 2.2.3 输入字段

**基本信息**
- 婚姻状态（单身 / 已婚或同居伴侣）
- 年龄（18-60）

**学历**
- 最高学历（无学历 / 高中文凭 / 1年制专科 / 2年制专科 / 学士/3年制 / 两个或以上证书含3年制 / 硕士/专业学位 / 博士）

**语言能力**
- 第一官方语言：语言测试类型（IELTS / CELPIP / TEF / TCF）+ 4项成绩（听说读写）→ 自动转换为 CLB
- 第二官方语言（可选）：同上

**工作经验**
- 加拿大工作经验年数（0 / 1 / 2 / 3 / 4 / 5年及以上）
- 海外工作经验（0 / 1-2年 / 3年及以上）

**配偶信息**（婚姻状态为已婚时显示）
- 配偶最高学历
- 配偶第一官方语言（CLB或测试成绩）
- 配偶加拿大工作经验年数

**附加分项**
- 是否持有省提名（是/否）
- 工作 Offer 类型（无 / NOC 00 / 其他 NOC 0、A、B）
- 加拿大学历（无 / 1-2年制 / 3年制或以上学位）
- 法语能力（无 / CLB 7+法语且英语低于CLB 5 / CLB 7+法语且不限英语水平）
- 在加拿大的兄弟姐妹（是/否）
- 持有认证行业资质证书（是/否）

#### 2.2.4 输出展示

- **总分**：大字体显示，配色根据分数区间变化（红/橙/绿）
- **分项详情**：可展开的分组列表，显示每个因素的得分和满分
- **评估建议**：基于分数给出简短建议文字

#### 2.2.5 IELTS → CLB 转换对照表

| CLB | 听力 | 阅读 | 写作 | 口语 |
|-----|------|------|------|------|
| 10+ | ≥8.5 | ≥8.0 | ≥7.5 | ≥8.0 |
| 9   | 8.0  | 7.0  | 7.0  | 7.5  |
| 8   | 7.5  | 6.5  | 6.5  | 7.0  |
| 7   | 6.0  | 6.0  | 6.0  | 6.0  |
| 6   | 5.5  | 5.0  | 5.5  | 5.5  |
| 5   | 5.0  | 4.0  | 5.0  | 5.0  |
| 4   | 4.5  | 3.5  | 4.0  | 4.0  |

CELPIP：与 CLB 直接对应（CELPIP 9 = CLB 9）

### 2.3 F2 — BCPNP 分数计算器

#### 2.3.1 适用流程

BC 省提名计划技能移民注册系统（SIRS），适用于以下项目：
- 技术工人（Skilled Worker, SW）
- 国际毕业生（International Graduate, IG）
- 初级和半熟练工人（Entry Level & Semi-Skilled, ELSS）

#### 2.3.2 评分结构（SIRS）

| 因素 | 最高分 |
|------|--------|
| 工作 Offer（工资水平 + NOC 级别） | 250 |
| 工作所在地区 | 30 |
| 工作经验 | 50 |
| 学历 | 50 |
| 英语语言能力 | 40 |
| 适应性加分 | 30 |
| **总计** | **450** |

#### 2.3.3 工作 Offer 详细评分

**工资（相对 BC 省中位薪资 ~$37/小时）**

| 工资区间 | 分数 |
|---------|------|
| 中位薪资 200%+ | 200 |
| 中位薪资 150-199% | 180 |
| 中位薪资 120-149% | 160 |
| 中位薪资 100-119% | 120 |
| 中位薪资 80-99% | 80 |
| 中位薪资 60-79% | 40 |
| 低于中位薪资 60% | 0 |

**NOC 职业等级**

| NOC 等级 | 加分 |
|---------|------|
| TEER 0 或 1（管理/专业） | +50 |
| TEER 2 或 3（技术/行政） | +30 |
| TEER 4 或 5（劳工/初级） | 0（仅 ELSS 适用） |

#### 2.3.4 输入字段

- 申请流程（SW / IG / ELSS）
- 工作 Offer 时薪（$）
- NOC 职业代码 / TEER 等级
- 工作所在地区
- 相关工作经验年数
- 最高学历
- 英语 CLB 分数（或 IELTS 成绩）
- 适应性因素（在 BC 工作/学习经历、配偶状况等）

#### 2.3.5 输出展示

- 总 SIRS 分数及分项
- 与近期邀请门槛对比
- 资格判断（符合/不符合各流程基本要求）

### 2.4 F3 — OINP 资格评估器

#### 2.4.1 功能描述

安大略省提名计划（OINP）没有统一的积分排名系统，各流程有不同的资格标准。本功能为用户提供各主要流程的资格评估。

#### 2.4.2 评估流程列表

| 流程 | 核心要求 |
|------|---------|
| 人力资本优先（HCP） | 在 EE 池中，CRS ≥ 400，NOC TEER 0-3，1年以上工作经验 |
| 雇主担保 - 外国工人 | Ontario 有效工作 Offer，NOC 0/A/B，2年以上工作经验 |
| 雇主担保 - 国际学生 | Ontario 高校毕业，NOC 0/A/B，Ontario 工作 Offer |
| 硕士毕业生流程 | Ontario 大学硕士学位（需近2年毕业） |
| 博士毕业生流程 | Ontario 大学博士学位（需近2年毕业） |
| 法语技术工人 | 在 EE 池中，法语 CLB 7+，工作经验1年+ |
| 紧缺技能流程 | 特定 NOC 职业，Ontario 工作 Offer |

#### 2.4.3 输入字段

- 是否在 EE 池中（是/否）
- CRS 分数（如在 EE 池中）
- 最高学历及毕业学校所在地
- 在 Ontario 工作/学习经历
- 工作 Offer（是/否，及 NOC 等级）
- 工作经验年数
- 法语水平（CLB）

#### 2.4.4 输出展示

- 可能符合的流程列表（绿色✓ / 灰色○）
- 每个流程的缺口分析
- 下一步行动建议

### 2.5 F4/F5/F6 — 邀请历史记录

#### 2.5.1 功能描述

展示各项目近一年的邀请轮次数据，帮助用户了解历史趋势。

#### 2.5.2 数据字段

| 字段 | 说明 |
|------|------|
| 日期 | 邀请发放日期 |
| 类型 | 邀请轮次类型（CEC / FSW / PNP / 技术类等） |
| 最低分 | 本轮最低邀请分数 |
| 邀请数量 | 本轮邀请人数 |

#### 2.5.3 展示形式

- **折线图**：历史最低邀请分数趋势（Chart.js 实现）
- **数据表格**：可排序的详细记录表
- **统计摘要**：最近一次、最高分、最低分、平均分

#### 2.5.4 数据声明

> 注意：历史数据仅供参考，基于公开资料整理，请以各官方网站最新数据为准。

## 三、非功能需求

### 3.1 技术要求

| 要求 | 规格 |
|------|------|
| 前端技术 | 纯 HTML5 / CSS3 / Vanilla JavaScript（无框架依赖） |
| 图表库 | Chart.js (CDN) |
| 部署方式 | 静态文件，支持直接打开 index.html |
| 浏览器兼容 | Chrome 90+, Firefox 88+, Safari 14+, Edge 90+ |
| 响应式 | 支持桌面（1200px+）和平板（768px+）布局 |

### 3.2 性能要求

- 页面首次加载时间 < 3 秒（无网络请求）
- 分数计算响应时间 < 100ms（实时计算）
- 图表渲染时间 < 1 秒

### 3.3 可用性要求

- 所有界面文字使用简体中文
- 表单输入有实时验证和错误提示
- 计算结果可复制/打印
- 支持键盘导航

## 六、免责声明

本工具仅供参考，不构成法律或移民建议。计算结果基于公开的评分规则整理，实际申请结果以加拿大移民局（IRCC）、BC省提名计划（BC PNP）及安大略省提名计划（OINP）官方决定为准。建议结合持牌移民顾问（RCIC）的专业意见进行申请规划。
